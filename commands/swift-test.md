# Swift Test Command

Run Swift tests with code coverage reporting.

## Usage

```
/swift-test
/swift-test --coverage
/swift-test --parallel
/swift-test UserServiceTests
```

## Options

- `--coverage` - Generate coverage report
- `--parallel` - Run tests in parallel
- `--filter` - Filter tests by name
- `--retry` - Retry failed tests
