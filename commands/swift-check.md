# Swift Check Command

Comprehensive Swift project analysis and validation.

## Usage

```
/swift-check
/swift-check --strict
```

## Checks Performed

1. **Code Quality**
   - SwiftLint violations
   - Code formatting
   - Naming conventions

2. **Build Health**
   - Compilation warnings
   - Deprecation warnings
   - Type safety issues

3. **Dependencies**
   - Outdated packages
   - Security vulnerabilities
   - Unused dependencies

4. **Best Practices**
   - Optionals usage
   - Memory management
   - Concurrency safety
