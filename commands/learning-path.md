# Learning Path

Get personalized learning recommendations based on your goals, experience level, and learning style.

## Command Usage

```
/learning-path [goal] [level] [timeframe]
```

## Available Learning Paths

### By Role (65+ Options)
- Frontend Developer, Backend Developer, Full-Stack Developer
- DevOps Engineer, Site Reliability Engineer
- Data Scientist, Machine Learning Engineer, AI Engineer
- Mobile Developer (Android/iOS/Flutter/React Native)
- Product Manager, Engineering Manager, Tech Lead
- Security Engineer, QA Engineer, Technical Writer

### By Level
- **Beginner:** Complete foundational knowledge
- **Intermediate:** 3-6 months development experience
- **Advanced:** 6-12 months in role
- **Expert:** 12+ months, looking to specialize

### By Timeframe
- **Fast Track:** 3-6 months (MVP skills)
- **Standard:** 6-12 months (professional level)
- **Deep Dive:** 12-24 months (mastery)
- **Expert:** 24+ months (specialization)

## Quick Start Examples

```
/learning-path frontend-developer beginner 6-months

/learning-path data-scientist intermediate 12-months
```

## Personalization Factors

Recommendations consider:
- Your current skills and experience
- Target role and goals
- Available time per week
- Learning style (visual, hands-on, theoretical)
- Career stage and timeline
