# Skill Assessment

Evaluate your knowledge across different technology areas and identify skill gaps for targeted improvement.

## Command Usage

```
/skill-assessment [domain] [optional: detailed]
```

## Assessment Domains

### Technology Areas
- **Frontend:** HTML, CSS, JavaScript, TypeScript, React, Vue, Angular
- **Backend:** Python, Java, Go, Node.js, API Design, System Design
- **Databases:** SQL, PostgreSQL, MongoDB, Redis, Data Design
- **DevOps:** Docker, Kubernetes, AWS, Linux, CI/CD, Infrastructure
- **Data & AI:** ML, Deep Learning, Data Engineering, LLMs, Analytics
- **Mobile:** Android, iOS, Flutter, React Native
- **Architecture:** Design Patterns, System Design, Security
- **Soft Skills:** Communication, Leadership, Testing, Documentation

## Assessment Types

### Quick Assessment (5-10 minutes)
- Basic knowledge check
- Identifies clear gaps
- Quick feedback
- Good for overview

### Detailed Assessment (15-30 minutes)
- In-depth evaluation
- Practical exercises
- Detailed feedback
- Personalized recommendations

### Hands-On Challenge (1-2 hours)
- Real code challenges
- Project-based assessment
- Performance metrics
- Detailed analysis

## Assessment Format

### Multi-Choice Questions
```
Question: What is the time complexity of binary search?
A) O(n)
B) O(log n) ✓
C) O(n²)
D) O(1)

Explanation: Binary search divides search space by half each iteration
```

### Practical Coding
```
Challenge: Write a function to find duplicate elements
Test cases provided
Auto-grading with feedback
```

### System Design
```
Design a social media feed system

Your design:
- Database schema
- API endpoints
- Caching strategy
- Scalability approach
- Trade-offs analysis
```

### Conceptual Understanding
```
Explain these concepts:
- SOLID principles
- Microservices architecture
- Event-driven systems
- Design patterns
```

## Skill Levels Explained

### Beginner (0-20%)
- Basic understanding
- Simple projects only
- Needs significant guidance
- Recent learner

### Novice (20-40%)
- Understands fundamentals
- Can build with guidance
- Some practical experience
- 1-3 months experience

### Intermediate (40-60%)
- Solid understanding
- Build independently
- Good practices
- 3-12 months experience

### Advanced (60-80%)
- Deep knowledge
- Complex problem solving
- Teach others
- 12-24 months experience

### Expert (80-100%)
- Mastery level
- Innovation focus
- Mentor junior developers
- 24+ months experience
