## Learning Prerequisites Chains

### Web Development Path
```
1. HTML (foundation)
   └─> CSS (styling)
       └─> JavaScript (interactivity)
           ├─> React (component library)
           ├─> Vue (progressive framework)
           └─> Angular (full framework)
```

### Type-Safe Development Path
```
1. JavaScript (must learn first)
   └─> TypeScript (adds types)
       ├─> React + TypeScript
       ├─> Vue + TypeScript
       └─> Angular (TypeScript-first)
```

### Design System Path
```
1. Design Fundamentals (required)
   ├─> Frontend Knowledge (recommended)
   │   ├─> HTML
   │   ├─> CSS
   │   └─> Component Libraries
   └─> Design System Roadmap
       ├─> Visual Elements
       ├─> Component Design
       ├─> Governance
       └─> Tooling
```

### UX Design Path
```
1. Design Fundamentals (foundation)
   └─> UX Design Roadmap
       ├─> Research Methods
       ├─> Design Thinking
       ├─> Interaction Design
       ├─> Visual Design
       └─> Testing & Validation
```

## Key Topics at a Glance

### HTML (120+ topics)
Essential knowledge:
- Semantic markup
- Forms and validation
- Media embedding
- Accessibility
- SEO fundamentals

### CSS (102 topics)
Essential knowledge:
- Selectors and specificity
- Box model
- Flexbox and Grid
- Responsive design
- Animations and transitions

### JavaScript (100+ topics)
Essential knowledge:
- Variables and data types
- Functions and scope
- Async programming (promises, async/await)
- DOM manipulation
- ES6+ modules

### TypeScript (102 topics)
Essential knowledge:
- Type system (primitives, unions, intersections)
- Interfaces and type aliases
- Generics
- Classes and inheritance
- Advanced types and utilities

### React (102 topics)
Essential knowledge:
- Components and JSX
- Hooks (useState, useEffect, useContext)
- State management (Context, Redux)
- Routing (React Router)
- Data fetching and testing

### Vue (115 topics)
Essential knowledge:
- Template syntax and directives
- Component registration
- Reactive data and computed properties
- Composition API vs Options API
- Pinia state management
- Vue Router

### Angular (140+ topics)
Essential knowledge:
- Components and modules
- Dependency injection
- RxJS observables
- Reactive forms
- Services and HTTP client
- Route guards and lazy loading

### Design System (100+ topics)
Essential knowledge:
- Design language and principles
- Visual design elements (colors, typography, spacing)
- Component architecture
- Design tokens
- Documentation and governance
- Accessibility guidelines

### UX Design (100+ topics)
Essential knowledge:
- User research methods
- Information architecture
- Wireframing and prototyping
- Interaction design
- User testing and validation
- Design tools (Figma, Adobe XD)
